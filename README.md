# News App - Stay Up-to-Date with the Latest News

This is a news application that enables users to stay up-to-date with the latest news. The application was built using Next.js, TypeScript, Jest, and Material UI to provide a modern and user-friendly experience.

## Front End

The front-end of the application was developed with mobile-friendliness in mind, incorporating Material UI for a polished and intuitive design. Users can easily navigate the app and view the latest news articles.

## Backend

The backend of the application includes an API call to retrieve news articles from the News API. Users can access the latest news articles, covering a variety of topics and categories.

## Technologies

The news application was built using the following primary technologies:

- `Next.js`: A popular framework for building server-side rendered React applications.
- `TypeScript`: A statically typed superset of JavaScript that provides optional type checking and improved development experience.
- `Jest`: A popular JavaScript testing framework used for testing React components and applications.
- `Material UI`: A library of React components that implement Google's Material Design for a modern and consistent user interface.

## Installation

To use the news application, follow these steps:

1. Clone the repository to your local machine using the command line interface.

2. Install the required dependencies by running the `npm install` command.

3. Start the application by running the `npm run dev` command.

4. Navigate to `http://localhost:3000` in your web browser to view the application.
